Installation
========================================

From Source
------------------------------

.. code-block:: sh

    $ git clone https://github.com/wdv4758h/jwconv
    $ cargo install --path jwconv/cli



Usage
========================================

.. code-block:: sh

    $ jwconv hana
    はな
    $ jwconv tourikenyuuki
    とうりけにゅうき
